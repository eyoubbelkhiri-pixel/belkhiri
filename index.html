GitHub Copilot passe en accès gratuit dans VS Code
2 000 complétions/mois inclus, choix entre GPT-4o et Claude 3.5 Sonnet…

GitHub lance Copilot Free : 2 000 complétions et 50 messages/mois dans VS Code, avec choix entre Claude 3.5 Sonnet et GPT-4o.

Résumé
L'IA générative rend les développeurs 55 % plus productifs selon Kyle Daigle, COO de GitHub. Mais risque de dépendance pour les juniors sans bases solides.

ANSSI & BSI : bonnes pratiques pour les assistants IA
Les agences française et allemande publient un guide conjoint sur les risques…
Résumé
L'ANSSI et le BSI publient "AI Coding Assistants" : code non vérifié, fuite de données, dépendances vulnérables. Ne jamais déployer du code IA sans relecture.

Tout code généré en PHP ou C# doit être relu et audité. Une requête SQL mal générée peut introduire une injection directement.
