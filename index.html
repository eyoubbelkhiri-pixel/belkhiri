L'IA générative rend les développeurs 55 % plus productifs selon Kyle Daigle, COO de GitHub. Risque de dépendance pour les juniors sans bases solides.

L'ANSSI et le BSI publient "AI Coding Assistants" : code non vérifié, fuite de données, dépendances vulnérables. Ne jamais déployer sans relecture.
